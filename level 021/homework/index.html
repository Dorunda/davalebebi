სტატია 1

JavaScript (JS)[a] is a programming language and core technology of the Web,
alongside HTML and CSS. It was created by Brendan Eich in 1995.[6] As of 2025,
overwhelming majority of websites (98.9%) uses JavaScript for client side webpage behavior.[10]

სტატია 2

Web browsers have a dedicated JavaScript engine that executes the client code. These engines are also utilized
some servers and a variety of apps. The most popular runtime system for non-browser usage is Node.js.[11]

გვერდითი ინფორმაცია

The ECMAScript standard does not include any input/output (I/O), such as networking, storage,
or graphics facilities. In practice, the web browser or other runtime system provides JavaScript APIs for I/O.
